## **Решение**

Ссылка на репозиторий: _<https://github.com/slw0000/slw0000.github.io>_

Решением данной лабораторной работы является данный сайт (url: _<https://slw0000.github.io>_), на котором есть как персональная информация (портфолио), так и в будущем будут повяляться отчёты по лабораторным работам

Сайт был реализован с помощью _MkDocs_ и сервиса _GitHub Pages_, для бесплатного хоста и деплоя статических сайтов.

С помощью внутренних настроек была сделана многостраничность, навигация. Также, с помощью дополнения Materials for MkDocs сайт был достаточно тонко кастомизирован

Тема `material` была выбрана из-за своей невероятно обширной кастомизации, а также совместимости с огромным количество плагинов (что потом может пригодиться, если сайт будет расширяться как персональное портфолио, например). Цветовая палитра выбиралась на глаз автора.

Главный файл сайта, в котором хранятся все настройки и кастомизация ***mkdocs.yml***:

```yml
site_name: Портфолио / Лабы

theme:
  name: material

  palette:
    scheme: slate
    primary: black
    accent: indigo

  font:
    code: Roboto Mono

  features:
    - navigation.tabs
    - navigation.sections
    - content.code.copy

markdown_extensions:
  - pymdownx.highlight:
      anchor_linenums: true
      line_spans: __span
      pygments_lang_class: true
  - pymdownx.inlinehilite
  - pymdownx.snippets
  - pymdownx.superfences

repo_url: https://github.com/slw0000

nav:
  - Главная: index.md
  - Об авторе: author.md
  - Лабораторные работы:
      - Лабораторная 1: labs/lab-01.md

```
